# 📱 Croma Mobile Sales Analysis Dashboard | Power BI

## 📌 Project Overview
This project is an interactive **Mobile Sales Analysis Dashboard** developed using **Power BI** for analyzing mobile sales performance at Croma.

The dashboard provides business insights into sales trends, customer behavior, payment preferences, product performance, and regional sales distribution through interactive visualizations and filters.

---

## 🎯 Objective
The objective of this dashboard is to:

- Monitor overall mobile sales performance
- Analyze brand-wise and model-wise sales
- Understand customer purchasing patterns
- Compare city-level sales contribution
- Evaluate payment method preferences
- Track customer ratings and transaction trends

---

## 📊 Dashboard Features

### KPI Metrics
- 💰 Total Sales
- 📦 Total Quantity Sold
- 🛒 Total Transactions
- 👥 Total Customers
- ⭐ Average Customer Rating
- 📈 Average Sales Value

### Interactive Filters
- Month Filter
- Brand Filter
- City Filter

### Visualizations
- Sales Distribution Map
- Total Sales by Mobile Model
- Payment Method Analysis
- Transactions by Day
- Brand Performance Table
- Customer Ratings Breakdown
- City-wise Sales Analysis

---

## 🛠️ Tools & Technologies Used

- Power BI

---

## 📈 Key Insights

- Delhi generated the highest sales contribution.
- UPI emerged as one of the most preferred payment methods.
- Apple and Samsung delivered strong revenue performance.
- Customer ratings remained consistently positive.
- Sales performance varied across months and cities.

---

## 🚀 How to Use

1. Clone the repository
2. Open `.pbix` file in Power BI Desktop
3. Refresh dataset if required
4. Explore dashboard using filters
